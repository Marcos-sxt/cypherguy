🔐 SEGURANÇA
━━━━━━━━━━━

Challenge-Response Authentication:
1. Gera challenge único (timestamp)
2. Envia para Secure Element via NFC
3. SE chip assina com Ed25519 (256-bit)
4. Private key NUNCA sai do chip
5. Apenas assinatura retorna
6. Signature verificada → Auth OK

Card ID como User ID:
• Cada cartão = ID único
• Impossível falsificar (hardware-backed)
• Usado em todas operações backend
• Multi-device support
